Struktura przechowująca dane dla wątku. 

Struktura zawiera informacje o przedziale pracy danego wątku\+:
\begin{DoxyItemize}

\item sum, w którym gromadzona jest częściowa suma,
\item step, czyli szerokość jednego prostokąta do metody prostokątów.
